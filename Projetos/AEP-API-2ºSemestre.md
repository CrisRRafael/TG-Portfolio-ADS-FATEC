A SPC Brasil, foi a empresa parceira do API do 2º semestre do curso de Análise e Desenvolvimento de Sistemas.

A SPC Brasil é uma empresa que *" oferece soluções a consumidores e empresários com o objetivo de facilitar a realização de negócios por meio de informação confiável.* "(fonte:https://www.spcbrasil.org.br/institucional/spc-brasil)

A SPC Brasil  *"estimula os brasileiros a fazer uso consciente e sustentável do dinheiro por meio da educação financeira e empodera os consumidores por meio de iniciativas como o Cadastro Positivo."* (fonte:https://www.spcbrasil.org.br/institucional/spc-brasil)

 A SPC Brasil tem o propósito de ser o elo de confiança entre as pessoas para a realização de negócios.

E visando esse propósito  e com a finalidade de criar um produto que agregasse valor a empresa a  equipe desenvolveu o Auxílio Emergencial Positivo.

O Auxílio Emergencial Positivo é  um serviço de proteção ao score positivo do consumidor, que a SPC Brasil poderá oferecer para quem é optante pelo cadastro positivo e que está adimplente com as suas contas.

O produto desenvolvido foi um aplicativo desktop para os Assistentes de Atendimento verificarem a viabilidade de inclusão do consumidor no Auxilio Emergencial Positivo.

O programa do Auxílio Emergencial Positivo surgiu através do compromisso da SPC Brasil tem com as pessoas, pelo compromisso constante de ser o elo de confiança para consumidores e empresas.

### Tecnologias Utilizadas

Para o desenvolvimento deste projeto a equipe optou por usar as seguintes tecnologias:

- Python - para o tratamento dos dados, bem como desenvolvimento do sistema
- Biblioteca  Tkinter para desenvolvimento da interface
- Outros recursos e bibliotecas foram utilizados para funções específicas (Pandas, Messagebox, Filedialog, Datetime, Xlrd).
